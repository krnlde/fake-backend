# fake-backend

[![Greenkeeper badge](https://badges.greenkeeper.io/krnlde/fake-backend.svg)](https://greenkeeper.io/)

## Install
`> npm install`
or
`> yarn`

## Run
`> npm start`

## Run and hotreload (via [nodemon](https://github.com/remy/nodemon))
Useful when developing APIs.

`> nodemon server.js`
